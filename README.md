# FrontEnd UI

It's basically an UI made using Nextjs.

## Tech Stack

**FrontEnd:** Nextjs, Reactjs, tailwind css, TypeScript.

## Installation

1. Clone my-project with Cloning from github

```bash
  git clone <repo-link>
```
2. Install the Dependencies

```bash
npm install
```

## Starting
Start the porject :

```bash
  npm run dev
```
